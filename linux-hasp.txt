[b]Виртуальная машина для Hyper-V основанная на Debian 10 (pycelle beta version)[/b]
Автор виртуалки: Dragonim
Потребляемые ресурсы: 256 Mb RAM & 5 Gb HDD
Данная виртуальная машина раздаёт в локальной сети 50 клиентских лицензий и позволяет прокинуть одну серверную лицензию для 64 битного сервера 1С. Для того чтобы прокинуть лицензию необходимо установить на целевом компьютере USB Redirector. Данная программа бесплатна для linux, для windows имеется взломанная версия.
После импорта виртуальной машины необходимо зайти на неё и поменять адрес, маску и шлюз локальной сети
$sudo nano /etc/network/interface
Пароль для пользователя user и root стандартный
После изменения адреса необходимо перезагрузить виртуальную машину
$sudo reboot
